# Guides

Collection publique de guides et tutoriels techniques.

Site : [https://alferann.github.io/guides/](https://alferann.github.io/guides/)

## Structure du depot

```
guides/
├── assets/          # CSS et ressources statiques
├── guides/          # Guides individuels (un dossier par guide)
├── .github/         # Workflows GitHub Actions
├── index.html       # Page d'accueil
└── LICENSE          # CC-BY-4.0
```

## Contribuer

1. Travailler sur la branche `dev`
2. Creer une branche `feature/` ou `fix/` depuis `dev`
3. Ouvrir une PR vers `main` pour publication
